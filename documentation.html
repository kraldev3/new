Installation

Shaderzone installs in under three minutes on any modern Windows 10 or 11 PC. You don't need administrator rights for the default install path.

  1. Download Shaderzone-Setup.exe from the home page.
  2. Run the installer and accept the EULA.
  3. Choose your preferred install location (default is %LocalAppData%\Shaderzone).
  4. Click Install. The installer will auto-detect your Roblox installation.
  5. When finished, hit Launch — that's it.
Heads up: Shaderzone does not modify any Roblox files. It runs as a separate background process and applies effects at the rendering layer.

First launch

On first launch, Shaderzone scans your GPU and runs a 5-second benchmark to recommend the optimal preset. You can override this recommendation at any time from the tray icon.

Open the dashboard from Win + Shift + S or by right-clicking the tray icon. Pick a preset, then start Roblox normally.

Hotkeys

  • Win + Shift + S — Open the dashboard
  • F8 — Toggle Shaderzone on/off in-game
  • F9 — Cycle through presets
  • F10 — Take a screenshot (saved to ~/Pictures/Shaderzone)
  • Ctrl + F8 — Show the performance HUD

Presets

Each preset tunes lighting, color grading, reflections and post-processing for a different look.

  • Realistic — Natural lighting, accurate colors, soft shadows. The default preset.
  • Cinematic — High contrast, film grain, anamorphic bloom. Best for screenshots and videos.
  • Anime — Saturated colors, cel-shaded outlines, dreamy bloom.
  • Performance — Minimal effects optimized for low-end GPUs and 144Hz+ setups.
  • Night Vision — Boosted brightness in dark scenes without losing detail.
  • Pro Pack — 50+ additional presets curated by the team. Optional upgrade.

Settings reference

Every preset can be tweaked from the Advanced tab.

  • Bloom intensity — How strongly bright pixels bloom (0.0–2.0).
  • Exposure — Overall image brightness (-2.0 to +2.0 EV).
  • Reflection quality — Off / Low / Medium / High / Ultra.
  • Volumetric fog — Atmospheric fog with light scattering.
  • Color grading LUT — Apply a custom .cube color grading file.
  • Sharpening — Contrast-aware sharpening (0–100).

Config file

For advanced users, all settings are stored in a JSON config file at:

%LocalAppData%\Shaderzone\config.json

Example:

{
  "activePreset": "cinematic",
  "hotkeys": {
    "toggle": "F8",
    "cyclePreset": "F9"
  },
  "render": {
    "reflectionQuality": "high",
    "bloomIntensity": 1.2,
    "exposure": 0.1,
    "volumetricFog": true,
    "sharpening": 35
  }
}

Troubleshooting

Roblox crashes after enabling Shaderzone

Make sure your graphics driver is up to date. Switch the render API in Roblox to DirectX 11 (Settings → Performance Mode → Manual).

No visual change after starting Roblox

Verify the tray icon shows Active. Some games launched through other launchers (custom clients, sandboxes) are not detected.

Performance drops on low-end GPUs

Switch to the Performance preset and disable reflections in the Advanced tab.

Note: If a game has its own anti-cheat that explicitly blocks third-party overlays, Shaderzone may be temporarily disabled in that experience for safety.

Uninstalling

Open Windows Apps & features, find Shaderzone in the list, and click Uninstall. All files in %LocalAppData%\Shaderzone are removed, and the system PATH is cleaned up automatically.
